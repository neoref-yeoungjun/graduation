# H.Log(충청북도 평생교육 플랫폼)
> 충청북도 지역에서 운영되는 각 기관별 흩어져 있는 강좌목록들을 하나로 규합 ∙ 분류하여 리스트(요일, 지역, 분야별)로 보여주며 사용자가 희망하는 강좌에 대한 자세한 정보를 찾고, 강좌가 열리는 기관에 대한 지도를 보여주는 안드로이드 애플리케이션
